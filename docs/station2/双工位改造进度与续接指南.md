## 3. 已完成工作

### 3.1 架构骨架（原阶段 1～2，已适配第二工位专用仓）

- `StationProfile`、`config/station_profiles/station2_*.ini`
- 9 个 PLC 触发器 → `TaskHandlerRegistry` + Handler 分发
- `enable*` 门控（LoadGrasp / UnloadCalc / PoseCheck 等）

### 3.2 第一工位剥离与精简（2026-06-16）

- 删除 `third_party` 算法库、`modules/tracking`、vision 适配层、`point_cloud_processor`
- 精简 `state_machine`、移除 `algorithmBypassEnabled` 与 PCL/OpenCV 构建链
- 合并扫描路径为 `station2_placeholder.json`
- HMI 保留并恢复 `cmd.set_bevel_recipe` / `cmd.debug_trigger_inspection` 协议占位

### 3.3 默认配置

```ini
# config.ini
[Station]
stationId=2
profileIni=config/station_profiles/station2_cylinder_semi.ini
scanPathsConfigPath=config/scan_paths/station2_placeholder.json
defaultWorkMode=MODE_CYLINDER_SEMI
```

切换封头/成品模式：改用 `station2_endcap.ini` 或 `station2_semi_finished.ini`（扫描路径暂共用占位 JSON）。

## 5. 待办阶段（续接入口）

### 阶段 A：第二工位扫描主链路（优先）

**目标**：`Trig_ScanSegment` 按 `scan_paths` 与工位模式执行真实多传感器采集。

**软件任务**

- 扩展 `station2_*.json`（`segmentKind`、段数、是否需要 2D 等）
- 实现 `ScanSegmentHandler`（替换 Res=8）
- `event.scan.finished` / `event.bundle.captured` 与 PLC Res 对齐

**参考**：[`工作流程-AI-Agent参考.md`](./工作流程-AI-Agent参考.md)、时序图 2.2～2.4

---

### 阶段 B：第二工位综合检测

**目标**：`Trig_Inspection` 跑检测算法并写 PLC + 推 HMI。

**软件任务**

- 实现 `InspectionHandler`
- 填充 `InspectionResult` / `InspectionMeasurement`（当前仅 `qualityCode`）
- 调用 `HmiTcpServer::publishInspectionResult`

---

### 阶段 C：吊装 / 安全 / 伸缩杆（硬件就绪后）

| 模块 | 说明 |
|------|------|
| `CollisionMonitorService` | 激光雷达防碰 |
| `HoistAssistService` | 吊装测距 + 2D |
| `TelescopicRodService` | 伸缩步进 |
| `RollerBedService` | 滚轮步进 |

`enableTelescopicScan` / `enableHoistAssist` / `enableCollisionMonitor` 绑定模块创建。

---

### 阶段 D：协议定稿与投产
