## 第二工位 — 核心时序图

依据 **`2 工作流程(动作+数据).pdf`**，按三种工件模式（封头 / 圆筒半成品 / 半成品·成品）提炼 **5 张** 核心图。

| 编号 | 文件 | 对应模式 | 说明 |
|------|------|----------|------|
| 2.1 | [系统安全与并行监控](./station2/2.1_系统安全与并行监控时序图.md) | 全部 | AI / 光栅 / 门锁 / 心跳，与主流程并行 |
| 2.2 | [封头模式主流程](./station2/2.2_封头模式主流程时序图.md) | `MODE_END_CAP` | 上料 3D → 抓取 → 扫描 → 检测 → 下料 3D |
| 2.3 | [吊装模式扫描与综合检测](./station2/2.3_吊装模式扫描与综合检测时序图.md) | 圆筒 / 成品 | 有料确认 → 外部多路径扫描 → 综合检测 |
| 2.4 | [圆筒伸缩杆内部扫描](./station2/2.4_圆筒伸缩杆内部扫描时序图.md) | `MODE_CYLINDER_SEMI` | 20 cm 纵缝步进 + 滚轮 10° 环缝 |
| 2.5 | [吊装辅助与碰撞监测](./station2/2.5_吊装辅助与碰撞监测时序图.md) | 吊装相关 | 测距定位、伸缩杆/吊装雷达防碰 |

> **阅读顺序建议**：2.1（安全）→ 按工件选 2.2 或 2.3+2.4 → 2.5（吊装场景并行参考）。

## 两工位对照（简表）

| 能力 | 第一工位 | 第二工位 |
|------|----------|----------|
| 上下料 | 机械臂 + 顶架 3D（IPC 解算抓取/下料点） | 封头：机械臂 + 上下料 3D；圆筒/成品：吊装 |
| 扫描执行 | IPC 驱动梅卡 + 海康多相机，PLC `Trig_ScanSegment` | PLC 编排扫描机械臂/转台/伸缩杆，IPC 采集与解算 |
| 检测员确认 | 流程内嵌 | 显式「是否运行」弹窗（上位机） |
| 内部扫描 | 无 | 圆筒模式：伸缩杆 + 滚轮架 |
| 防碰辅助 | IPC 雷达信号回 PLC | 伸缩杆雷达 + 1/2 路吊装雷达 + 激光测距定位 |
