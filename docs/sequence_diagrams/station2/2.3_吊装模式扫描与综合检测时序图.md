# 2.3 吊装模式扫描与综合检测时序

**工位**：兰铀第二工位  
**工件模式**：`MODE_CYLINDER_SEMI` / `MODE_SEMI_FINISHED`（圆筒半成品、半成品·成品）  
**依据**：`2 工作流程(动作+数据).pdf` 模式 B/C 主扫描段  
**说明**：两种模式共用「扫描工位有料 → 多路径扫描 → 综合检测」框架；圆筒模式在 2.4 中补充伸缩杆内部扫描。

```mermaid
sequenceDiagram
    autonumber
    participant O as 检测员
    participant P as PLC
    participant H as 上位机
    participant I as 工控机(IPC)
    participant S as 扫描机械臂
    participant T as 转台

    P->>P: 自检合格（含吊装/伸缩杆复位检查）
    P->>S: 扫描工位点位1：2D AI 有料判断
    I->>P: 是否有料(Modbus TCP)
    alt 无料
        P->>H: 报警 + 弹窗
    else 有料
        P->>H: 「是否运行」确认
        O->>H: 确认运行
        H->>P: 检测员选择

        loop 外部多点位扫描
            P->>T: 转台转到指定角度(RS485/Modbus)
            P->>S: 扫描机械臂到目标位
            P->>I: 开始拍照/扫描
            I->>I: 采集本路径图像/点云
            I->>P: 结束拍照/扫描
        end

        P->>S: 扫描机械臂复位
        P->>T: 转台复位

        P->>I: 开始缺陷 + 编号 + 3D 检测
        I->>P: 检测是否顺利执行
        alt 检测未完成
            P->>H: 报警；要求重新吊装测量
        else 检测完成
            P->>P: 本件流程结束
        end
    end
```
